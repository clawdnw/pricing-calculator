## Workflows & Automations

1. **Morning Brief Pipeline:** Agent proactively sends daily brief with news, weather, overnight work summary, and content ideas
2. **Newsletter Drafting:** Agent learned newsletter schedule from conversation, proactively writes drafts
3. **Vibe Code → PR → Review:** Text idea → agent codes it → pushes to GitHub → creates PR → sends link for review

5. **Email Forwarding:** Forward emails to Henry's Gmail → Henry responds/schedules/acts on them
6. **Second Brain:** Text any thought → agent categorizes into folders (ideas/inbox/research)
7. **To-do List Management:** Text Henry → get today's to-do list → "choose one and help me with it"
8. **Overnight Building:** "Build something cool for me tonight" → wake up to new app
9. **Thumbnail Pipeline:** AirDrop template → text Henry → agent uses AI image tool to edit
10. **Kanban Task Management:** Add tasks to backlog → assign to Henry → Henry works through them

## Tips, Tricks & Best Practices

1. **Name your agent and give it a logo** — makes interaction more fun and personal
2. **Use a separate computer/VPS** — zero guardrails means it could mess with personal files, emails, messages
3. **Give it its own email account** — don't use your personal Gmail
4. **Give it its own accounts for everything** — separation of concerns
5. **Brain dump everything about yourself first** — interests, job, daily/weekly work, tools, favorite teams, everything
6. **Set up a morning brief with "ideas for today"** — have it suggest tasks based on what it knows about you
7. **Lean on the agent for task generation** — "What can you do for me?" rather than always directing
8. **Build a kanban/mission control board** — track agent tasks, create backlogs
9. **Set up a second brain** — replace Notion with agent-managed file organization
10. **Treat it like onboarding a human employee** — "take them to lunch," get to know each other
11. **Before bed: "build something cool tonight"** — maximize overnight hours
12. **Personality matters A LOT** — Opus feels human; robotic responses kill the experience
13. **Start cheap, scale up** — base Mac Mini or VPS first, upgrade if needed
14. **External SSD for storage** — cheaper than buying a bigger Mac Mini
